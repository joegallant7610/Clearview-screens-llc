## How to Deploy to GitHub

### Option 1: GitHub Desktop (Easiest)
1. Download GitHub Desktop app
2. Create new repository called "clearview-screens-website"
3. Add all files to the repository folder
4. Commit and publish to GitHub
5. Go to repository Settings > Pages
6. Enable GitHub Pages from main branch
7. Your site will be live at: `yourusername.github.io/clearview-screens-website`

### Option 2: GitHub Web Interface (Mobile Friendly)
1. Go to github.com and create new repository
2. Name it "clearview-screens-website"
3. Make it public
4. Upload these files:
   - current-site.html (rename to index.html)
   - mural-painting.jpg
   - mural-turtle-room.jpg
   - mural-mermaid.jpg
   - FB_IMG_1665740706770 copy.jpg
   - FB_IMG_1665740706770.jpg
5. Go to Settings > Pages
6. Select "main" branch and click Save
7. Wait 2-3 minutes for deployment

## Alternative: Quick Deploy with Netlify

1. Go to netlify.com
2. Drag and drop your folder with all files
3. Site goes live instantly with a random URL
4. You can customize the URL in settings
